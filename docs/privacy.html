Política de Privacidade Este aplicativo foi desenvolvido para funcionar totalmente offline e não coleta, armazena, compartilha ou transmite quaisquer dados pessoais ou do dispositivo. Coleta e uso de informações O app não coleta informações pessoais, tais como nome, e-mail, localização, identificadores do dispositivo ou qualquer outro dado sensível. Nenhum dado é enviado para servidores externos. Conexão com a Internet O aplicativo não utiliza conexão com a internet para nenhuma funcionalidade. Dados de crianças Como nenhum dado é coletado, o app é seguro para todos os públicos, incluindo crianças menores de 13 anos. Segurança Embora o app não colete informações, adotamos boas práticas de desenvolvimento para garantir que nenhuma informação do usuário seja acessada indevidamente pelo aplicativo. Alterações nesta Política Esta política pode ser atualizada ocasionalmente. Alterações serão publicadas nesta mesma página.
